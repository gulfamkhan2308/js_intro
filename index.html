JavaScript Assignment

1. What is JavaScript ?

JavaScript is a programming language used to create interactive and dynamic content on websites.

2. How do you include JavaScript in an HTML file?

JavaScript can be included in two ways:

  • Internal JavaScript: Written inside <script> tags within an HTML file.
    • <script>
    alert('Hello, JavaScript!');
</script>
  • External JavaScript: Linked as a separate .js file.
    • <script src="script.js"></script>

3. What is console.log() used for?

The console.log() function is used to display messages in the browser's console, mainly for debugging.

4. Write a JavaScript script to print "Hello, JavaScript!" in the console.

5. What is an alert box in JavaScript ?

An alert box is a pop-up message that displays a notification to users.

6. Write a script to display an alert box with the message "Welcome to JavaScript!".

7. Name some popular JavaScript frameworks and their uses.

  • React: Used for building user interfaces, especially for single-page applications.
  • Angular: A full-fledged front-end framework developed by Google.
  • Vue.js: A progressive framework for building UI and single-page applications.

8. What is the MERN stack?

The MERN stack consists of four technologies used for full-stack web development:

  • MongoDB: NoSQL database
  • Express.js: Backend framework for Node.js
  • React: Frontend JavaScript library
  • Node.js: JavaScript runtime environment

1. What are variables in JavaScript?

Variables are used to store and manage data. We can think of them as containers for values like numbers, strings, or objects.

Declaring Variables in JavaScript

JavaScript provides three ways to declare variables:

  1. var (old way, avoid using)
  2. let (modern and recommended for changeable values)
  3. const (for values that should not change)

2. What are the rules for nameing variables?

Rules for Naming Variables

  1. Variable names must start with a letter, _, or $.
  2. They cannot start with numbers.
  3. They are case-sensitive (myVar and myvar are different).
  4. They should not be JavaScript reserved keywords (var, let, const, etc.)

3. What are the different data types in JavaScript?

JavaScript has eight main data types, which are categorized into primitive and non-primitive (reference) types.
A data type defines the kind of values a variable can hold

A. Primitive Data Types (Stored directly in memory, Stores only a single value)

These are immutable (cannot be changed)

1. String (Text)

enclosed in single ('), double ("), or backticks (`)

let greeting = "Hello, World!";
console.log(greeting); // Output: Hello, World!

2. Number (Whole numbers & decimals)

Represents numbers (integers & floating-point).

let age = 30;
let price = 99.99;
console.log(age, price);

3. Boolean

Represents true or false.

let isRaining = false;
let isSunny = true;
console.log(isRaining);

4. Null

Represents an empty or unknown value

let car = null;
console.log(car); // Output: null

5. Undefined

A variable that has been declared but not assigned a value.

let temperature;
console.log(temperature); // Output: undefined

B. Non-Primitive Data Types (Stores multiple values)

1. Objects

Used for collections of key-value pairs

let student = {
name: "Ali",
age: 20
isEnrolled: true }
; console.log(student.name); // Output: Al

2. Arrays

Ordered lists for storing data store multiple value

let numbers = [1, 2, 3, 4];
console.log(numbers);
// Output: [1, 2, 3,4]

4. What is the difference between == and ===?

  • == (loose equality) checks only values and allows type conversion.
  • === (strict equality) checks both value and type.

9. What is the difference between alert, prompt, and confirm?

  • alert("Message"): Displays a message with an OK button.
  • prompt("Enter value"): Asks for input and returns a string.
  • confirm("Are you sure?"): Returns true for OK and false for Cancel.
